Windows 7

Installation guide (VM or hardware).

Requirements
  • VM: 2 GB RAM (minimum), 25–60 GB disk
  • Media: Windows 7 ISO
Steps
  1. Create VM; attach Windows 7 ISO (or boot from USB on hardware).
  2. Boot → Install Now.
  3. Choose Custom install.
  4. Partition and format (NTFS). Select target partition.
  5. Installation completes; create user and computer name.
  6. Install chipset/video/network drivers.
  7. Run Windows Update (if it still works in your environment).
Tips

In a VM, install Guest Additions/VMware Tools for smoother performance.
