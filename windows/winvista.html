Windows Vista

Installation guide (VM recommended).

Requirements
  • VM: 2 GB RAM (minimum), 20–40 GB disk
  • Media: Windows Vista ISO
Steps
  1. Create VM; attach Vista ISO.
  2. Boot; choose language/time/keyboard.
  3. Click Install Now; enter key (or skip if allowed).
  4. Choose edition; accept license.
  5. Custom Install → select disk/partition (create if needed).
  6. Installation runs; reboots multiple times.
  7. Complete first-boot setup (username, password, updates).
Notes

Vista needs more resources; performance is better with 2+ cores and 3–4 GB RAM if available.
