Windows 95

Installation guide (best installed in a Virtual Machine).

Requirements
  • VM: 64–128 MB RAM (more can work), 500 MB–2 GB disk, 1 CPU
  • Media: Windows 95 CD/ISO + boot floppy image (often needed)
Steps (VM approach)
  1. Create new VM (Other/DOS or Windows 95 profile if available).
  2. Attach a Windows 95 boot floppy image to Floppy (or boot ISO that includes DOS tools).
  3. Boot the VM to DOS.
  4. Partition the disk:
    • Run: FDISK
    • Create Primary DOS partition, set it Active, then reboot.
  5. Format the disk:
    • Run: FORMAT C: /S
  6. Mount the Windows 95 CD/ISO.
  7. Start setup:
    • Go to CD drive (ex: D:)
    • Run: SETUP
  8. Follow prompts (name, key, components).
  9. After install, set display/sound drivers if available; Guest Additions may not fully work.
Troubleshooting
  • “Missing HIMEM”: adjust VM settings; use correct boot disk.
  • CD-ROM not detected in DOS: load MSCDEX + CD driver in config.sys/autoexec.bat.
