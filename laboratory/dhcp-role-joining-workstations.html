DHCP Role and Joining Workstations in Domain

Add the DHCP Server role, configure scopes for male and female students, and join client workstations to the domain.

Add Roles

DHCP Server

Scopes Configuration
  • Number of male students per year and section is the host requirement with the IP block 10.0.0.0 /8
  • Number of female students per year and section is the host requirement with the IP block 172.16.0.0 /16
  • Server: Last Usable IP Address
  • Exclude: First & Last (10 usable IP Addresses)
  • Lease Time: 16 hours
  • Router: Server IP Address
  • DNS: Server IP Address (8.8.8.8)
Client OS Setup
  1. Create client OS: 1 Windows XP (LASTNAME-XP) and 1 Windows 7 (LASTNAME-WIN7).
  2. Connect the two (2) client OS to your Windows Server Domain using DHCP.
    • Windows XP IP address must belong to Class A network.
    • Windows 7 IP address must belong to Class B network.
  3. Login 5 student accounts per section per OS.
