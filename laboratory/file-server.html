File Server

A file server is a centralized storage system within a network that allows multiple users to store, access, and manage files and data. It provides a shared location for documents, media, and other resources, enabling collaboration and efficient data management.

Overview

File servers often include features like access control, user permissions, and backup capabilities to ensure data security and availability. They can be implemented using dedicated hardware or software solutions and are commonly used in businesses, schools, and organizations to streamline file sharing, reduce redundancy, and maintain centralized control over critical data.

Instructions

Add roles: File Services, and perform the following:

1. Folder: Public

  • Permissions: Read/Write
  • Users: All
  • Files: All file formats except *.exe, *.com, *.vb, *.vbs

2. Folder: VB.Net

  • Permissions: Read/Write
  • Users: All
  • Files: *.zip, *.rar

3. Folder: C_Programs

  • Permissions: Read/Write
  • Users: All
  • Files: *.c, *.cpp

4. Folder: Surname_Section

  • Permissions: Full Control
  • Users: Domain User
  • Files: *.c, *.cpp, *.pdf, *.rar, *.docx, *.ppt, *.dwg

5. Folders: PHOTO_Male & PHOTO_Female

  • Permissions: Read/Write
  • Users: Based on Users Gender
  • Files: *.jpg, *.gif, *.png
